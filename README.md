# Wandern Berg - moderne, statische Website

Neuaufbau fuer **Wandern Berg / Michael Kleemann** mit Fokus auf ruhiges, naturbezogenes Design, gute Lesbarkeit und klare Struktur.

## Projektstruktur

- `index.html` - Startseite mit Bereich "Die naechsten Reiseziele"
- `ueber-wanderreisen.html` - Ueberblick und Reisephilosophie
- `reisen/index.html` - Reiseuebersicht
- `reisen/*.html` - einzelne Reise-Unterseiten
- `handicap-team.html` - Informationen zum Handicap-Team
- `kontakt.html` - Kontaktdaten
- `impressum.html` - Impressum (Platzhalter)
- `datenschutz.html` - Datenschutz (Platzhalter)
- `components/header.html` - wiederverwendbare Navigation inkl. Dropdown
- `components/footer.html` - wiederverwendbarer Footer
- `assets/css/styles.css` - globales, responsives Styling
- `assets/js/main.js` - Komponenten laden, Navigation, Dropdown, aktiver Link

## Inhalte austauschen

- **Texte**: direkt in den jeweiligen `.html`-Dateien anpassen.
- **Kontaktdaten**: vor allem in `kontakt.html` und im Footer-Inhalt.
- **Reisen**: neue Karten in `reisen/index.html` und neue Detailseiten in `reisen/` anlegen.
- **Bilder**: in `assets/css/styles.css` sind derzeit Platzhalter-URLs (Unsplash). Diese durch eigene Bilder ersetzen.
